# KILJU CALCULATOR
This project is a simple application for calculating the approximate % of Kilju after n days fermenting. It can help you decide how much sugar, and how many litres of water you should use.
## Features:
* Calculate ABV
* Return approximate price per litre - calculated with Google Gemini 2.5 Flash API.
